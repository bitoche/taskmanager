# Планировщик задач в виде календаря на неделю.

##  Настройка синхронизации через ЯД:
В проекте присутствует синхронизация файла базы данных через Яндекс.Диск. 

Это решение актуально, т.к. проект подразумевает однопользовательское использование, но пользователь может пользоваться сразу несколькими гаджетами.

1. На вашем Я.Д. необходимо создать директорию:
`/taskservice/tech`
2. Перейти на сайт Я.Полигон ([ссылка](https://yandex.ru/dev/disk/poligon/)), и получить ваш токен по кнопке `Получить OAuth-токен`.
3. Полученный токен необходимо добавить в `taskmanager/taskservice/.env`, в переменную `REMOTE_STORAGE_TOKEN`

Далее, если проект ранее был поднят без использования облака, необходимо в интерфейсе нажать на кнопку "Сохранить в облако", после чего файл будет сохранен на ваш ЯД.

## Разработка:
- поднять всё в первый раз:
```bash
docker compose up --build -d
```

- применить изменения во фронте:
```bash
docker compose up --duild -d webui
```

- применить изменения в taskservice:
```bash
docker compose restart taskservice
```
